Google Translate Widget — Demo

This page shows the free Google Website Translator widget (client-side). Use the dropdown to translate the page into the languages listed. You can also use the quick buttons below.

Sample content

Welcome to this demo. The Google Translate widget will translate this page's content into the language you select. This widget is a client-side convenience and not the Cloud Translation API.

Note: The free widget script is provided by Google and requires network access to translate.google.com. Some environments or ad-blockers may prevent it from loading.

Paragraphs to translate

This example contains multiple paragraphs, headings, and inline elements so you can verify how the translation appears across different parts of the page.

Try the buttons above or pick a language from the dropdown.

Notes: The Google Website Translator widget is different from the paid Cloud Translation API. The widget translates the visible page only. For production or server-side translations, use the Cloud Translation API (requires billing & API key).
